# Minio

## Overview

The Minio It wraps the official stable [Minio chart](https://github.com/helm/charts/tree/master/stable/minio).
Find more information about the Minio chart in the chart's [documentation](chart/minio/README.md).

## Details

The minio addon contains two plans:
- micro
- entrerpise

It is a show case how to define chart values in a plan.
